Проект: API для управления детскими товарами

# Описание проекта:

Этот проект представляет собой **REST API** для управления каталогом детских товаров.  
Разработан с использованием **FastAPI** — современного высокопроизводительного фреймворка для создания API на Python.

### Основные возможности:
-  Создание товаров (POST)
-  Просмотр всех товаров (GET)
-  Просмотр одного товара по ID (GET)
-  Обновление товара (PUT)
-  Удаление товара (DELETE)
-  Очистка всей базы (DELETE)
-  Автоматическая документация (Swagger UI / ReDoc)
-  Валидация данных через Pydantic
